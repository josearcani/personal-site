# Dotnet commands

```
- dotnet ef migrarions add CreateInitial
- dotnet ef database update
- dotnet new xunit
- dotnet new classlib
- dotnet new webapi
- dotnet new ignore
- dotnet new sln --name <solution_name>
- dotnet sln <solution_name.sln> add <path_of_csproj>
- dotnet add <path_of_csproj> reference <path_of_csproj>
```
